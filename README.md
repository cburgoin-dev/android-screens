# Mobile Programming Activity: Android Screens

This repository contains a small Android project made for a **Mobile Programming class activity**.

The goal of the assignment was to practice building basic user interfaces using **Kotlin and Jetpack Compose**. The project includes simple **Login** and **Register** screens designed with Compose components.

This project is part of a classroom exercise to get familiar with layout structures, UI components, and basic styling in Android development.
